# Cyber Awareness Website

A comprehensive, interactive website dedicated to promoting cybersecurity awareness and educating users about common cyber threats and protective measures.

## Features

### 🛡️ Core Sections

1. **Home/Hero Section**
   - Eye-catching introduction with animated shield
   - Clear call-to-action buttons
   - Welcoming design to engage visitors

2. **Common Cyber Threats**
   - Malware
   - Phishing
   - Weak Passwords
   - Unsecured Networks
   - Social Engineering
   - Privacy Violations
   - Expandable cards with detailed protection tips

3. **Essential Safety Tips**
   - 8 critical cybersecurity best practices
   - Numbered tips for easy reference
   - Actionable guidance for each recommendation

4. **Interactive Quiz**
   - Test your cybersecurity knowledge
   - Immediate feedback with scoring
   - Encourages learning and retention

5. **Resources Section**
   - Curated list of password managers
   - Recommended VPN services
   - Essential security tools
   - Educational resources

6. **Contact Section**
   - Contact information
   - Message form for inquiries
   - Multiple ways to get in touch
